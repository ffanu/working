üöÄ Setup

Prerequisites

  • .NET SDK 9.0+
  • Node.js 18+
  • MongoDB 6.0+

Quick Start

1. Start MongoDB

# macOS
brew services start mongodb-community

# Linux
sudo systemctl start mongod

# Verify
mongosh

2. Start Backend

cd backend/InventoryAPI
dotnet restore
dotnet run

Backend runs on http://localhost:5236

3. Start Frontend

cd frontend
npm install
npm run dev

Frontend runs on http://localhost:3009

‚úÖ Access: Open http://localhost:3009 in your browser

üîß Troubleshooting

MongoDB Connection Failed

  • Verify MongoDB is running: mongosh
  • Check connection string in appsettings.json
  • Default: mongodb://localhost:27017

Backend Not Starting

  • Check .NET version: dotnet --version
  • Restore: dotnet restore
  • Check port 5236 is available

Frontend Issues

  • Check Node version: node --version
  • Reinstall: rm -rf node_modules && npm install
  • Verify API URL in src/lib/config.ts
  • Check browser console (F12) for errors

Port Conflicts

# Find process using port
lsof -i :3009
lsof -i :5236
lsof -i :27017

# Kill process
kill -9 <PID>

Quick Debug Commands

# Test backend
curl http://localhost:5236/api/products

# Test frontend
curl http://localhost:3009

# Check Docker
docker-compose ps
docker-compose logs
